> **Специализированная студия для создания мини-гемблинг игр (слоты, рулетка, карточные игры) на Flutter + Flame.** Проект управляется и разрабатывается через систему координированных ИИ-агентов Claude Code.

Вы — **творческий директор и продюсер**. Агенты реализуют вашу идею от математического баланса до сочных (juicy) анимаций!

---

## ✨ Ключевые особенности

- 🧠 **Управляемые агенты**: Целая виртуальная студия узкоспециализированных ИИ-сотрудников.
- 🚀 **Быстрый старт**: Генерация прототипов от идеи до рабочего кода одной командой.
- 🎯 **Mobile-First**: Оптимизировано для Flutter Impeller (iOS/Android).
- 🎲 **Математика и RTP**: Интегрированные инструменты для симуляции баланса и выплат.

---

## 🛠 Технологический стек

- **Движок**: Flutter 3.27+ / Flame 1.18+
- **Язык**: Dart 3.6+ (null-safe, sealed classes, pattern matching)
- **Специализация**: Мини-гемблинг игры (mobile-first)
- **Рендеринг**: Flutter Impeller (iOS/Android), Skia (desktop)

---

## 🚀 Быстрый старт

Просто используйте slash-команды в вашем терминале Claude Code для управления студией:

- 🟢 `/start` — Ориентация: с чего начать прямо сейчас.
- 💡 `/brainstorm` — Интерактивная генерация концепта гемблинг-игры.
- ⚡ `/auto-idea` — Автономная генерация готовой идеи (без вопросов).
- 🏗️ `/autocreate` — *Zero-to-playable*: концепт + ассеты + код за одну команду!

### Полный пайплайн создания игры:
`Концепт` ➔ `Математика` ➔ `Дизайн` ➔ `Прототип` ➔ `Ассеты` ➔ `Код` ➔ `Баланс` ➔ `Релиз`

## 🤖 Команда Студии (Агенты)

Ваша студия укомплектована профессионалами высшего класса:

### 👑 Tier 1 — Дирекция
* **`creative-director`**: Общее видение, концепт, творческие решения.

### 🎲 Tier 2 — Специалисты гемблинга
* **`rtp-mathematician`**: Расчёт RTP, весов символов, симуляция вероятностей.
* **`gambling-game-designer`**: GDD для слотов (барабаны, линии, бонусы, символы).
* **`slot-programmer`**: Логика вращения, RNG, комбинации.
* **`juice-artist`**: VFX, партиклы, анимации — делает игру «сочной».

### 💻 Tier 3 — Разработка и QA
* **`lead-programmer`**: Архитектура, ревью кода.
* **`ui-programmer`**: Экраны Flutter, HUD, панель ставок.
* **`sound-designer`**: Аудио эффекты и музыка.
* **`qa-tester`**: Тест-кейсы, edge cases, проверка RNG.
* **`release-manager`**: Подготовка к выпуску.

👉 Мощная команда: `/team-gambling [описание]` — собирает нужных специалистов для комплексной задачи.

## 🎮 Архетипы мини-игр

Студия поддерживает автоматическое создание следующих игровых архетипов:

| ID | Название | Жанр | Особенности механики |
|:--:|----------|------|---------------------|
| **A** | *Неоновый Спин* | 3-барабанный слот | Near Miss система, каскадная остановка |
| **B** | *Счастливое Колесо* | Колесо фортуны | Сектора с разными выплатами, множители |
| **C** | *Покер Экспресс* | Видео-покер | 5 карт, Hold функция, двойная ставка |
| **D** | *Фруктовая Буря* | 5-барабанный слот | Каскадные символы (Avalanche), Free Spins |
| **E** | *Скрэтч Делюкс* | Скретч-карты | Мгновенные выигрыши, 3 попытки |
| **F** | *Рулетка Неон* | Мини-рулетка | Европейская рулетка, внешние/внутренние ставки |

*Совет: Введите `/auto-idea --archetype [A-F]`, чтобы сгенерировать игру определенного жанра.*

## 📋 Практики работы и качество

- **Баланс и RTP:** Запустите `/balance-check` для симуляции 1,000,000 спинов и проверки математики.
- **Подготовка к выходу:** Команда `/release-checklist` проведет проверку всей gambling-специфики перед релизом.
- **Существующие проекты:** Используйте `/continue-project` или `/add-feature [фича]` для развития готовых игр.
