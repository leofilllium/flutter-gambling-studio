# Быстрый Старт

Добро пожаловать в **Flutter Gambling Studio**! Это пространство настроено для быстрого
прототипирования и разработки мини-гемблинг игр (слоты, рулетка, карточные игры).

Здесь вы выступаете в роли **Директора Студии**, а AI-агенты — ваша команда.
Ваша главная задача — принимать решения, а всю рутину команда возьмет на себя.

> **ВАЖНО**: Всё общение со студией ведется исключительно на **русском языке**.

## 🚀 Как начать новую игру?

У вас есть два пути:

### Путь 1: Автоматический (Я хочу готовую игру)
Просто введите:
```bash
/autocreate
```
Студия сама выберет 1 из 6 архетипов (например, 3-барабанный неоновый слот), напишет дизайн, нарисует SVG, создаст код, настроит `pubspec.yaml` и подготовит проект к запуску.

### Путь 2: Ручной (Я хочу создать уникальный слот)

**Шаг 1. Идея**
```bash
/brainstorm
```
Вместе с агентом выберете тему, волатильность (частые мелкие выигрыши или редкие большие), и уникальную фичу (каскады, Wild-множители).

**Шаг 2. Разбор на компоненты**
```bash
/map-systems
```
Студия создаст план сборки вашей игры.

**Шаг 3. Детальный дизайн RTP**
```bash
/design-system rtp-weights
```
Тут подключается `rtp-mathematician`. Он рассчитает математическую модель выплат так, чтобы RTP был 95-97%.

**Шаг 4. Написание кода**
```bash
/team-gambling "Реализуй ядро слота по нашему концепту"
```
Запустится оркестрация `slot-programmer` (пишет логику) и `juice-artist` (делает крутые анимации).

## 👥 Ваша команда (Агенты)

Никогда не пытайтесь писать код слота сами. Просто вызывайте нужного специалиста:

| Специалист | Кого звать | Что делает |
|------------|------------|------------|
| Математик | `@rtp-mathematician` | Считает RTP, настраивает вероятности выпадания. |
| Геймдизайнер | `@gambling-game-designer` | Придумывает бонусные раунды, количество линий выплат. |
| Программист Слота | `@slot-programmer` | Пишет код барабанов на Flame, RNG генератор. |
| Художник Эффектов | `@juice-artist` | Делает игру "сочной": тряска камеры, монеты, фидбэк спина. |
| UI/UX | `@ui-programmer` | Делает кнопки Spin, Max Bet, панель баланса на Flutter. |
| Звуковик | `@sound-designer` | Добавляет звуки вращения, выигрышей и Near Miss. |

## 🛠 Полезные команды в процессе разработки

Если вам нужны картинки:
```bash
/generate-asset symbol вишня
/generate-asset ui spin-button
```

Проверить баланс математики (симуляция 1 миллиона спинов):
```bash
/balance-check
```

Добавить фичу в уже готовую игру:
```bash
/add-feature "Добавь Free Spins раунд"
```

Сделать паузу и продолжить завтра:
```bash
/continue-project
```

Готовы начать? Введите `/start` прямо сейчас.
