# Технические стандарты Гемблинг Студии

## Flutter + Flame 1.18.x

### Математика и RNG
- **КРИТИЧЕСКИ**: НИКОГДА не используйте `math.Random()`.
- **ВСЕГДА**: Используйте `dart:math` `Random.secure()` для:
  - Выбора символов на барабанах
  - Раздачи карт в покере
  - Остановки колеса рулетки / фортуны
  - Триггеров бонусных механик
- **Stateless Outcomes**: В момент запуска спина результат УЖЕ 100% вычислен. Вы знаете, будет ли это выигрыш *до* начала вращения первой визуальной катушки. Анимация просто "проигрывает" предопределенный сценарий. Важно для синхронизации клиента и сервера, а также корректного RTP.
- **RTP Tuning**: Все веса (weights) хранятся в `config.dart` в виде констант или берутся из JSON файла, сгенерированного `rtp-mathematician`.

### Flame API (1.18.x)

- Наследуйте главный класс от `FlameGame`.
- Коллизии всегда объявляем на `World`, а не на `FlameGame`:
  `class SlotWorld extends World with HasCollisionDetection {}`
- Используйте обновленную `CameraComponent`:
  `camera = CameraComponent(world: _world);`
- Никаких `.isPaused = true`. Используйте `GameState` (ValueNotifier/enum: Idle, Spinning, Win).

### Визуализаторы и Партикли

Для сочности гемблинг игр мы используем эффекты *ParticleSystemComponent*.
- При выигрышах > x20, спавните шквал золотистых частиц:
  `ParticleSystemComponent(particle: Particle.generate(count: 50, generator: ...))`
- Настройки эффекта (glow, drop shadow) реализуются через Flutter Overlay поверх Flame, так как во Flame сложные фильтры потребляют много ресурсов.

### Звук
- Используйте пакет `flame_audio` `^2.1.0`.
- Ограничивайте параллельное звучание: максимум 3 накладывающихся звука (например: 1 BGM loop, 1 Spins sound loop, 1 Effect Overlay).
- Монеты считаются с нарастанием pitch: `playbackRate` 1.0 -> 1.5.

### Ассеты SVG
- Для графики используется `flame_svg` `^1.10.0`.
- Ассеты генерируются через команду `/generate-asset`.
- Паттерн наименования:
  `background_X` (фоны)
  `sprite_X` (символы слотов, элементы)
  `ui_X` (кнопки, панели)
  `icon_X` (значки баланса, ставки)
