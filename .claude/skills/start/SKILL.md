# `flutter-gambling-studio` — Знакомство со студией

Привет! Вы находитесь в специализированной студии разработки мини-игр на **Flutter + Flame**.

Мы создаём мини-игры любого жанра: гемблинг, пазл, аркада, физика, казуальные, карточные/настольные.

> **Правило 1**: Всё общение со мной и агентами студии ведётся на **русском языке**.

### Главные команды для старта

| Команда | Что делает |
|---------|------------|
| `/brainstorm` | Пошаговая генерация идеи игры любого жанра. Выберем тему, механику и уникальную «сочность». |
| `/auto-idea` | Мгновенно генерирует полную концепцию игры (без вопросов) из каталога 24 архетипов A-X. |
| `/autocreate` | Собирает игру от концепции до готового Flutter-проекта за одну сессию. |
| `/continue-project` | Продолжить работу с игрой с точки остановки. Ваша текущая точка входа. |

### Команда специалистов

Вас обслуживают специализированные агенты:
- **`game-mathematician`** — баланс игры, вероятности, RTP (для гемблинга), кривые сложности.
- **`game-designer`** — проектирует игровые системы, уровни, механики.
- **`mechanics-programmer`** — пишет игровую логику, RNG, физику на Flame `1.18.x`.
- **`juice-artist`** — делает анимации и эффекты «сочными».

***

**С чего начнём?** Введите `/brainstorm` для интерактивного создания игры,
`/auto-idea` для мгновенной генерации концепта или `/autocreate` для быстрого старта.
