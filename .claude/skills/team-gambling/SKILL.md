# `team-gambling` — Оркестрация Студии

Запускает агентов в правильном порядке для реализации сложной фичи.

## Инструкция

1. Уточните задачу у пользователя: какая фича, есть ли готовый GDD?
2. Если GDD нет — вызовите `gambling-game-designer` для его написания.
3. Если нужна математика (вероятности выпадания фичи, её ценность) — вызовите `rtp-mathematician`.
4. Для реализации вызовите `slot-programmer` (Core Logic) и `juice-artist` (VFX анимации) в нужном порядке. Обязательно передайте им ссылки на GDD.
5. Предложите пользователю проверить результат когда всё готово.
