# `generate-asset` — Студия SVG Ассетов

Выполняет запросы на генерацию SVG файлов для игры.

1. Узнайте **цветовую палитру** (если не передана в аргументе, попросите убрать/добавить, если есть GDD - прочитать оттуда).
2. Выберите тип ассета:
   - `symbol`: 64x64 или 96x96 вишня, лимон, семерка, скаттер. Яркие, с градиентами (SVG `<linearGradient>`). Должны быть четкими на телефоне.
   - `ui`: кнопки Spin, Max Bet, рамки спинов с `<feDropShadow>` или `<feGaussianBlur>` для неонового свечения (glow).
   - `background`: полноэкранный (9:16 mobile) фон с абстрактным узором, сеткой, казино-эстетикой. (Тёмный).
3. Сохраните в `assets/images/sprites/` или `assets/images/ui/`. Обязательно добавьте путь в `pubspec.yaml` если папка новая.

> Не забывайте `<svg viewBox="0 0 W H" xmlns="http://www.w3.org/2000/svg">`.
